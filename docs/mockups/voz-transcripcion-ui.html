A · Aviso de fallo con campo de clave Recomendado

Cuando Gemini rechaza la clave, sale este panel. Pegas una clave nueva ahí mismo, “Guardar y reintentar”, y la nota se transcribe sin entrar a Preferencias. Botón OK para cerrar.

C · Fallback Gemini → OpenAI en Preferencias

El switch que pediste: si Gemini falla, usa ChatGPT (OpenAI) automáticamente. Así una clave mala no te deja sin transcripción si la otra sí sirve.
